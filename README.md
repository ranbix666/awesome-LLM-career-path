# Awesome LLM Career Path

An interactive roadmap and learning guide for engineers looking to break into LLM/AI engineering.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set your `GEMINI_API_KEY`:
   `cp .env.example .env.local`
3. Run the app:
   `npm run dev`
